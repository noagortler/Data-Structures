## 2. Linked List

### a. Reverse a Linked List
- **Input:** `[5] -> [3] -> [11] -> [65]`
- **Output:** `[65] -> [11] -> [3] -> [5]`

### b. Find the Middle of a Linked List
- **Input (odd):** `[5] -> [3] -> [11] -> [65] -> [100]`
- **Output:** `[11]`
- **Input (even):** `[5] -> [3] -> [11] -> [65]`
- **Output:** `[3]`

### c. Detect a Cycle
Return `true` or `false` if the Linked List contains a cycle.
- **Input:** `[5] -> [3] -> [11] -> [65] -> [100] -> [5]`
- **Output:** `true`
- **Input:** `[5] -> [3] -> [11] -> [65] -> [100] -> Null`
- **Output:** `false`

### d. Remove Duplicates
- **Input:** `[5] -> [3] -> [11] -> [65] -> [11]`
- **Output:** `[5] -> [3] -> [65]`
